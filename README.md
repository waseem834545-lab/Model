# Car Price Prediction App

This is a Streamlit app for predicting car prices using a pre-trained XGBoost model.  
Users can upload a CSV file with car features and actual prices to get predictions and visualizations.

## How to Deploy on Streamlit Cloud

1. Push the entire folder to GitHub.
2. Go to [Streamlit Cloud](https://streamlit.io/cloud) and create a new app.
3. Connect your GitHub repo, select branch and `streamlit_app.py`.
4. Deploy! The app will load the model and preprocessor automatically.
